The Vercel plugin handles incoming webhooks. Point your provider’s subscription URL at your Corsair HTTP handler (see [Overview](/plugins/vercel/overview) for setup context and the exact URL shape).

<Info>
**New to Corsair?** See [webhooks](/concepts/webhooks) and [hooks](/concepts/hooks).
</Info>

## Webhook map

- `deploymentCreated` (`deploymentCreated`)
- `deploymentError` (`deploymentError`)
- `deploymentSucceeded` (`deploymentSucceeded`)
- `projectCreated` (`projectCreated`)

## HTTP handler setup

```ts app/api/webhook/route.ts
import { processWebhook } from "corsair";
import { corsair } from "@/server/corsair";

export async function POST(request: Request) {
    const headers = Object.fromEntries(request.headers);
    const body = await request.json();
    const result = await processWebhook(corsair, headers, body);
    return result.response;
}
```

## Events

## Deployment Created

### Deployment Created

`deploymentCreated`

**Payload:** `unknown`


**`webhookHooks` example**

```ts
vercel({
    webhookHooks: {
        deploymentCreated: {
            before(ctx, args) {
                return { ctx, args };
            },
            after(ctx, response) {
            },
        },
    },
})
```
