## Posts

### create

`posts.create`

Create a new LinkedIn post for a member or organization.

**Risk:** `write`

```ts
await corsair.linkedin.api.posts.create({});
```

**Input**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `author` | `string` | Yes | Author URN. Use urn:li:person:... to post as a person (w_member_social) or urn:li:organization:... to post as an organization (w_organization_social). |
| `commentary` | `string` | Yes | The text commentary (body) of the post. |
| `visibility` | `PUBLIC \| CONNECTIONS \| LOGGED_IN_MEMBERS` | No | Who can see the post. Defaults to PUBLIC. |
| `lifecycleState` | `PUBLISHED \| DRAFT` | No | PUBLISHED to post immediately, DRAFT to save as a draft. |
| `isReshareDisabledByAuthor` | `boolean` | No | When true, other members cannot reshare this post. |
| `article_url` | `string` | No | Optional URL to attach as article content with the post. |
| `article_title` | `string` | No | Title for the attached article. Requires article_url. |
| `article_description` | `string` | No | Description for the attached article. Requires article_url. |
| `image_urn` | `string` | No | Optional image URN to attach as media. |



**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `id` | `string` | Yes | The URN of the newly created post. |



---
